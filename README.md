# Laravel API

## Build the project

### Composer

```
composer update
composer install
```

### Start local server
Be careful with DB settings!

#### Local server on Windows
```
php artisan serve
```
#### Local server on Linux
```
php artisan serve --host=192.168.13.50 --port=8000
```
